Audiorepr - A Python Audiolization Package
==========================================================


A python package to represent data using musical notes. Similar to visualization packages, ``audiorepr`` takes in data and audiolize the data.

Here is a demo using `a sample of covid19 data <https://gist.github.com/emptymalei/90869e811b4aa118a7d28a5944587a64>`_ in Europe. The audio is the result of this package while the video is made with GarageBand. (See the tutorials for the code.)
